Shows posts of a sepcific category in reverse order (oldest to newest) OR shows only a specific category in reverse order (oldest to newest) on the home page

== Description ==
Category Reverse Order is a plugin for WordPress which can reverse the order of posts for a specific category OR show only a specific category in reversed order on the home page.

== Installation ==
1. upload zip to your plugin-folder or install it from the WordPress Library
2. activate the plugin
3. go to settings page and chose the category you want to reverse
3.1 Choose if you want to display only the chosen category on the home page (checked means yes)
4. save
